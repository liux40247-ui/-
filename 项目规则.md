# 目标检测评估平台 - 项目规则文档

## 一、项目概述

本项目是一个基于Flask的目标检测模型评估平台，支持YOLOv5/YOLOv8系列模型的评估、对比和可视化分析。

## 二、技术栈

### 后端框架
- **Flask 3.0.0** - Python Web框架
- **Ultralytics 8.1.0** - YOLO模型库（支持YOLOv5/YOLOv8）

### 数据处理
- **NumPy 1.24.3** - 数值计算
- **OpenCV 4.9.0** - 图像处理
- **Pillow 10.1.0** - 图像操作
- **PyYAML 6.0** - 配置文件解析

### 前端技术
- **Bootstrap 5** - UI框架
- **Chart.js** - 图表可视化
- **jQuery** - AJAX交互

## 四、核心功能模块

### 4.1 数据集管理
- 支持图片批量上传
- 支持YOLO格式标注文件
- 自动统计类别分布
- 数据集增删查改

### 4.2 模型管理
- 支持YOLOv5系列：n/s/m/l/x
- 支持YOLOv8系列：n/s/m/l/x
- 模型自动下载（首次使用时）
- 动态模型加载/卸载

### 4.3 评估功能
- 单模型评估
- 多模型对比评估
- 评估指标：mAP、Precision、Recall、F1、FPS
- 可调节置信度和IoU阈值

### 4.4 可视化展示
- 指标对比图表
- 类别分布图
- 速度对比图
- 历史记录查看

## 五、API接口

### 数据集接口
```
GET  /api/datasets/              # 获取数据集列表
POST /api/datasets/upload-with-labels  # 上传数据集
GET  /api/datasets/<id>          # 获取数据集详情
DELETE /api/datasets/<id>        # 删除数据集
```

### 模型接口
```
GET  /api/models/list            # 获取模型列表
GET  /api/models/<id>/info       # 获取模型信息
POST /api/models/<id>/load       # 加载模型
```

### 评估接口
```
POST /evaluate/api/run           # 执行单模型评估
POST /evaluate/api/compare       # 执行多模型对比
GET  /evaluate/api/history       # 获取评估历史
```

### 结果接口
```
GET  /results/api/history        # 获取历史记录
GET  /results/api/<id>           # 获取结果详情
```

## 六、配置说明

### 模型配置 (models/model_config.yaml)
```yaml
models:
  yolo8n:
    name: "YOLOv8 Nano"
    type: "yolo"
    version: "v8"
    size: "n"
    weights: "yolov8n.pt"
    description: "YOLOv8最小版本"
    input_size: 640
    params: "3.2M"
```

### 应用配置 (config.py)
- 数据集路径：`static/datasets/`
- 最大文件大小：100MB
- 支持图片格式：jpg, jpeg, png, bmp, webp

## 七、开发命令

```bash
# 安装依赖
pip install -r requirements.txt

# 运行开发服务器
python app.py

# 访问地址
http://localhost:5000
```

## 八、使用流程

1. **上传数据集**：在数据集管理页面上传图片和标注文件
2. **选择模型**：在评估页面选择YOLO模型
3. **执行评估**：设置参数后点击开始评估
4. **查看结果**：查看mAP等指标和可视化图表
5. **模型对比**：选择两个模型进行对比分析

## 九、扩展模型

在 `models/model_config.yaml` 中添加新模型配置即可，无需修改代码：

```yaml
models:
  yolo11n:
    name: "YOLO11 Nano"
    type: "yolo"
    version: "v11"
    weights: "yolo11n.pt"
    input_size: 640
    params: "2.6M"
```

## 十、注意事项

- 模型首次使用时会自动下载（约10-50MB）
- 建议使用CPU模式运行，兼容性更好
- 数据集标注文件需与图片文件名对应
- 评估结果存储在SQLite数据库中
